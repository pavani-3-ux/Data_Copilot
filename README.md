# 🚀 Data Copilot

**Data Copilot** is an AI-powered analytics platform that enables users to interact with datasets using natural language. Upload your data, ask questions in plain English, generate visualizations, uncover business insights, and make smarter decisions without writing complex code.

---

## 📖 Overview

Data Copilot acts as your personal AI Data Analyst.

Whether you're a business analyst, student, data enthusiast, or decision-maker, Data Copilot helps you:

- Analyze datasets instantly
- Discover hidden trends
- Generate visual reports
- Create interactive charts
- Forecast future performance
- Gain actionable business insights

---

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/data-copilot.git
```

```bash
cd data-copilot
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux / Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Backend

```bash
uvicorn main:app --reload
```

---

## ▶️ Run Frontend

```bash
npm install
```

```bash
npm run dev
```
